All In One
"All in One" is a Frontend development project to show and test developer skills. It is developed with the following resources: HTML:5, CSS, Angular v.13, Bootstrap 5. In this development you can find the following: SCSS, Flexbox, Grid, Responsive, Lazy loading, Session Storage, Reactive Forms, Routes, Guards, Pipe, BEM Methology, SMACSS Methology, Gitflow.
